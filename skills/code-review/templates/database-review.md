# 数据库审查规则（所有语言通用）

## SQL 安全性审查

| 检查项           | 检测方法             | 重构建议       |
|------------------|----------------------|----------------|
| SQL注入风险      | 字符串拼接SQL        | 使用参数化查询 |
| 权限过度授权     | GRANT ALL PRIVILEGES | 最小权限原则   |
| 明文存储敏感信息 | 密码、密钥未加密     | 使用加密存储   |

## SQL 性能审查

| 检查项   | 检测方法           | 重构建议           |
|----------|--------------------|--------------------|
| 缺少索引 | 全表扫描查询       | 添加适当索引       |
| 索引滥用 | 过多或重复索引     | 合并索引、删除冗余 |
| 大表操作 | 无分区的全表操作   | 分区表、分批处理   |

## 表结构设计审查

| 检查项       | 检测方法           | 重构建议       |
|--------------|--------------------|----------------|
| 规范化不足   | 冗余数据、更新异常 | 实施第三范式   |
| 字段类型不当 | VARCHAR(255)滥用   | 选择合适类型   |
| 缺少约束     | 无外键、非空约束   | 添加完整性约束 |

## 数据库架构审查

| 检查项       | 检测方法       | 重构建议       |
|--------------|----------------|----------------|
| 单点故障     | 单数据库实例   | 主从复制、集群 |
| 备份策略不足 | 无定期备份     | 实施备份计划   |
| 监控缺失     | 无性能监控     | 添加监控告警   |
